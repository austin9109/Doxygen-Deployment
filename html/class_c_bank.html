Detailed Description

For user, bank needs to offer IBankService. For bank to bank communication while transfering, another bank needs to offer ICrossBankService for bank to transfer some cash. Thus we design bank to inherit IBankService and ICrossBankService.

When interact with bank API, user need to required for an access token first, then access bank's other method with this token. Bank will check if this token is valid and direct to corresponding account.
